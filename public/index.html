Image Processing Api Usage

Manualy enter your query parameters as following:

  • imageName: Should be equal to one of the three images found in the assets/full folder ex.RobinSharma-3840x2160.jpg

  • width: The width of the resized image ex.width=400

  • height: The height of the resized image ex.height=200

  • The full url: Should be something like this http://127.0.0.1:3000/resize?imageName=RobinSharma-3840x2160.jpg&width=600&height=400
